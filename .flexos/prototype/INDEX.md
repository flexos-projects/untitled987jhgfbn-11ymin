# prototype/

HTML prototypes. Standalone pages viewable in an iframe.

- Versioned: `{slug}-v{N}.html` (e.g. `home-v2.html`)
- Can exist without specs (drafts, explorations, variants)
- Link to spec via frontmatter when ready
- Share tokens via `design/tokens.md` → generated CSS
- Share mock data via `mock-data.js` (generated from database specs)
