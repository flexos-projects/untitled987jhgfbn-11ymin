# specs/

Per-item specifications. One file per thing.

**Types:**
- `page` — UI specs (subtypes: `route`, `component`, `layout`)
- `feature` — feature specs
- `database` — collection/schema specs
- `flow` — user journey specs

**Naming:** `NNN-{type}-{subtype}_{slug}.md`

**Examples:**
- `001-page-route_home.md`
- `002-page-component_swipe-card.md`
- `003-page-layout_default.md`
- `004-feature_auth.md`
- `005-database_users.md`
