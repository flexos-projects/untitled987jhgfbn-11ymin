# design/

Three files:

- **tokens.md** — colors, typography, spacing, radii, shadows, transitions. One flex_block per category.
- **components.md** — reusable HTML+CSS snippets. One flex_block per component. Copy-pasteable into prototypes.
- **layouts.md** — page layout patterns with SVG wireframes. One flex_block per layout.

All components use `var(--token)` references. Swap tokens, swap the look.
